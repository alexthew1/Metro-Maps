# MetroMaps 8.1 - Initial Release

**Welcome to MetroMaps**, the modern reimagining of the classic Windows Phone experience.

**Key Features**
*   **Authentic Design**: Experience the iconic "Metro" typography, pivot navigation, and fluid animations.
*   **Real-Time Navigation**: Turn-by-turn routing with voice-ready maneuvers and "Course Up" map orientation.
*   **Smart Search**: Find places, categories, and coordinates powered by Google Places.
*   **Multi-Mode Routing**: Directions for Driving, Walking, and Public Transit.
*   **Rich Details**: View photos, ratings, reviews, and contact info for businesses.
*   **Offline-Ready UI**: Designed to be lightweight and battery-efficient.

*Built with React Native & Expo.*
